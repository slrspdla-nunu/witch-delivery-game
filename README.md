# 🧹 마녀 배달부의 비상착륙

> 달빛 하늘 도시를 배경으로, 꼬마 마녀 **루나**가 고양이 **네로**와 함께 빗자루를 타고 의뢰 물품을 배달하는 모바일 웹 게임입니다.

빌드 과정 없이 `index.html`만으로 동작하는 정적 HTML/CSS/JavaScript 게임으로, 모바일 세로 화면(390×844)을 기준으로 디자인되었습니다.

---

## 🎮 플레이 방법

1. **첫 화면 터치** → 배경음악이 시작되고 메인 화면이 등장합니다.
2. **게임 시작** → 메인 메뉴로 진입합니다.
3. **의뢰하기** → 배달할 의뢰(케이크·고양이·마력 포션·드래곤 알)를 선택합니다.
4. **비행 & 배달** → 카운트다운 후 빗자루로 하늘을 날아 목적지까지 배달합니다.
   - **위/앞뒤 버튼**으로 고도와 속도를 조절합니다.
   - **까마귀**를 피하세요 — 부딪히면 체력(HP)과 **배달물 상태**가 깎입니다.
   - **아이템**(별사탕·모래시계·포션)을 모아 이득을 얻습니다.
   - **대시**(MP 소모, 가속)와 **보호막**(1회 충돌 방어) 스킬을 사용할 수 있습니다.
5. **결과** → 배달물이 무사하면 **성공**, 파손·시간 초과·체력 소진 시 **실패** 화면이 나옵니다.

### 게임 규칙 요약

| 항목 | 값 |
|---|---|
| 제한 시간 | 50초 |
| 까마귀 충돌 피해 | HP −18, 배달물 −12 |
| 대시 | MP −25, 2.8초 가속(×1.75), 쿨타임 5초 |
| 보호막 | MP −35, 4초 지속, 쿨타임 8초 |
| 성공 별점 | 남은 체력·충돌 횟수에 따라 1~3★ |

## 🖥️ 화면 구성

- **스플래시** — 로고·부제 등장, 첫 제스처에서 BGM 시작
- **메인 메뉴** — 상단 재화·프로필 HUD, 좌/우/하단 메뉴, 액션 메뉴(의뢰·모험 등)
- **의뢰 선택** — 2×2 카드로 의뢰·난이도·보상 표시 (메인 화면 비율로 고정)
- **비행(플레이)** — 가로형 스크롤 비행, HP·MP·시간·거리 HUD, 조작·스킬 버튼, 일시정지
- **배달 성공** — 별점·획득 보상, 반짝이/발광 연출, 루나 부양 애니메이션
- **배달 실패** — 내리는 비·차가운 발광 연출, 위로금 보상, 다시 도전하기
- **스토리 모드(모험)** — 배경 맵 위 스테이지 노드(클리어/현재/잠김)를 잇는 챕터 진행 화면

---

## ✨ 주요 기능

- 스플래시 → 메인 순차 등장 연출
- **PixiJS 메쉬**로 루나·네로에 바람에 흔들리는 정점 애니메이션 적용
- 물리 기반 비행 조작(가속·관성·경계 처리)과 장애물·아이템·스킬 시스템
- 성공/실패 결과 화면의 파티클(반짝이·비) 연출
- 스토리 모드 챕터 맵
- 첫 사용자 제스처 이후에만 재생되는 배경음악
- 컨테이너 쿼리(`cqw`/`cqmin`) 기반 비율 유지 UI

## 🚀 실행 방법

정적 파일이라 로컬 서버로 여는 것을 권장합니다(오디오·이미지 로딩 이슈 방지).

```bash
git clone https://github.com/slrspdla-nunu/witch-delivery-game.git
cd witch-delivery-game

# 아무 정적 서버나 사용 가능
python -m http.server 8000
```

이후 브라우저에서 `http://localhost:8000` 접속.

배포 버전(GitHub Pages): https://slrspdla-nunu.github.io/witch-delivery-game/

- Vanilla JavaScript
- [PixiJS](https://pixijs.com/) — 캐릭터 메쉬 변형 애니메이션

## 📝 개발 메모

- 모바일 세로(390×844)를 기준으로 설계, 컨테이너 쿼리로 비율을 유지합니다.
- 게임 로직은 `js/game.js`, 화면 전환·버튼 피드백은 `script.js`가 담당합니다.
- 배경음악은 브라우저 자동재생 정책에 맞춰 **첫 사용자 입력 이후**에만 재생됩니다.
- UI 이미지는 화면별로 `image/` 하위 폴더에 PNG 에셋으로 분리되어 있습니다.
